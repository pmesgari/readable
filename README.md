# Udacity React Developer Readable Project

This is the second project in the react developer curriculum offered by Udacity.
This project showcases usage of react with redux, to manage app state.

## Getting Started

To get started, both the client and server must be running.
For the client, fork or clone this repo and run the following commands:
```
npm install
npm run start
```
For the server, fork or clone [server](https://github.com/udacity/reactnd-project-readable-starter) and run the following commands:
``` 
cd api-server
npm install
node server
```
### Prerequisites

This project uses `npm` and `node`

## Built With

* [create-react-app](https://github.com/facebook/create-react-app) - The web framework used
* [react-redux](https://github.com/reduxjs/react-redux) - State management
* [Bootstrap](https://github.com/twbs/bootstrap) - UI and styling
